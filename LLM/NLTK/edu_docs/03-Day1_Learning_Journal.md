### 4. 어휘 다양성 (Lexical Diversity)
**개념:**  
어휘 다양성 = 고유 단어 수 / 전체 단어 수

**의미:**
- 높을수록: 다양한 어휘 사용
- 낮을수록: 반복적인 어휘 사용

**계산 예시:**
```python
diversity = len(set(text1)) / len(text1)
```
