# 어떻게 Foundation Model을 App으로 적용 시킬 수 있을까?



### 시작시 고려할 사항

* 이 AI App을 왜 만들어야 할까?
* 나의 App을 어떻게 평가할까? AI를 이용하여 자동적으로 평가할 수 있을까?
* Hallucination은  왜 발생할까? Hallucination을 어떻게 탐지하고 완화할 수 있을까?
* 프롬프트 엔지니어링의 모범 사례는 무엇일까?
* RAG는 왜 효과가 있을까? RAG를 수행하기 위한 전략은 무엇이 있을까?
* Agent란 무엇일까? Agent를 어떻게 만들고, 평가해야 할까?
* 모델을 언제 파인튜닝해야 할까? 언제 파인튜닝하지 말아야 할까?
* 데이터는 얼마나 필요할까? 데이터 품질은 어떻게 검증해야 할까?
* 모델을 더 빠르고, 저렴하고, 안전하게 만들려면 어떻게 해야 할까?
* App을 지속적으로 개선하기 위한 피드백 루프는 어떻게 만들까?
